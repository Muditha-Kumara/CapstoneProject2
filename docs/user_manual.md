1. Accessing the Application

The application can be accessed through a web browser.
Users can browse public pages without logging in, but certain features require authentication.

2. Creating an Account

To register as a new user:

Click Sign Up on the homepage.

Fill in required information such as name, email, and password.

Submit the form to create the account.

Use your email and password to log in.

3. Logging In

Existing users can log in by:

Clicking Login

Entering email and password

Submitting the form

If credentials are valid, the user is redirected to the dashboard.

4. Dashboard Overview

Once logged in, the dashboard displays:

Summary of actions available

Recent activity

Navigation links to different parts of the system

5. Submitting a Food Request

Users can request assistance using the Food Request Form:

Open the request form

Fill in the required information

Submit the request

The request is sent to the backend for processing

6. Donating as a Donor

Donors can choose to help by:

Viewing available requests

Selecting a request to fulfill

Confirming the donation

The system updates the request status accordingly.

7. Notifications

Users may receive notifications when:

Their request status is updated

A donor selects their request

Administrative actions occur

8. Logging Out

Users can log out using the Logout button located in the navigation bar.

9. Error Handling

If any error occurs:

Check that the form fields are filled correctly

Ensure a valid email format

Refresh the page if the dashboard fails to load

More troubleshooting is included in the troubleshooting guide.
